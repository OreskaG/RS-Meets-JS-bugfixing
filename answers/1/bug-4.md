# Баг 4 — Поздравление не появляется начиная с 10

## Что происходит

Когда счётчик доходит до 10, поздравление не показывается. Оно появляется только при 11 и выше.

## Почему так происходит

В функции `render` проверяется условие `count > 10` (строго больше), тогда как нужно `count >= 10`:

```js
// ❌ Баг
function render() {
  counterEl.textContent = count;
  if (count > 10) {
    statusEl.textContent = '🎉 Поздравляем, вы достигли 10!';
  }
}
```

При `count === 10` условие `10 > 10` ложно, поэтому поздравление не срабатывает.

## Как исправить

```js
// ✅ Исправление
if (count >= 10) {
  statusEl.textContent = '🎉 Поздравляем, вы достигли 10!';
}
```

## Как лучше

Когда в тексте сообщения написано «вы достигли 10», важно понимать что именно проверяется: `> 10` значит «больше 10», а `>= 10` значит «10 и больше». Всегда сверяй оператор сравнения с тем, что написано в сообщении — они должны совпадать.
