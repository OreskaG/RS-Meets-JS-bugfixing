# Баг 3 — Кнопка «Сброс» не обнуляет счётчик

## Что происходит

При нажатии на «Сброс» счётчик показывает `1` вместо `0`.

## Почему так происходит

В обработчике `resetBtn` переменной `count` присваивается `1` вместо `0`:

```js
// ❌ Баг
resetBtn.addEventListener('click', function () {
  count = 1;
  statusEl.textContent = 'Нажми «+», чтобы начать считать';
  render();
});
```

## Как исправить

```js
// ✅ Исправление
resetBtn.addEventListener('click', function () {
  count = 0;
  statusEl.textContent = 'Нажми «+», чтобы начать считать';
  render();
});
```

## Как лучше

Такие баги («почти правильно, но чуть-чуть не то») легко пропустить глазами. Помогает привычка после написания кода запускать его и проверять каждую кнопку вручную — именно с граничными значениями: 0, 1, максимум.
