# Баг 1 — Кнопка «+» уменьшает счётчик

## Что происходит

При клике на кнопку «+» значение счётчика уменьшается вместо того, чтобы увеличиваться.

## Почему так происходит

В обработчике кнопки `increment` стоит неверный оператор:

```js
// ❌ Баг
incrementBtn.addEventListener('click', function () {
  count = count - 1;
  render();
});
```

`count - 1` вычитает единицу. Оператор `-` перепутан с `+`.

## Как исправить

```js
// ✅ Исправление
incrementBtn.addEventListener('click', function () {
  count = count + 1;
  render();
});
```

Или короче — через оператор инкремента:

```js
count++;
```

## Как лучше

`count++` — стандартный идиоматичный способ увеличить число на 1 в JavaScript. Он читается быстрее, чем `count = count + 1`, и именно его чаще всего увидишь в реальном коде.
