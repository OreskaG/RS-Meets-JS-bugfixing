# Баг 1 — Кнопка «+» уменьшает счётчик

## Что происходит

При клике на кнопку «+» значение счётчика уменьшается вместо того, чтобы увеличиваться.

## Почему так происходит

В обработчике кнопки `increment` стоит неверный оператор:

```js
// ❌ Баг
incrementBtn.addEventListener('click', function () {
  count = count - 1;
  render();
});
```

`count - 1` вычитает единицу. Оператор `-` перепутан с `+`.

## Как исправить

```js
// ✅ Исправление
incrementBtn.addEventListener('click', function () {
  count += 1;
  render();
});
```

Или короче — через составное присваивание:

```js
count += 1;
```

## Как лучше

`count += 1` — понятная запись без путаницы суффиксной/префиксной формы, которая возникает с `count++`/`++count`.
