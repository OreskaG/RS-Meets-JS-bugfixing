# Баг 5 — Счётчик уходит в отрицательные числа

## Что происходит

При нажатии «−» счётчик опускается ниже нуля: −1, −2, −3...

## Почему так происходит

В обработчике кнопки «−» нет никакой проверки на минимальное значение. Код просто всегда вычитает 1:

```js
// ❌ Баг (нет защиты от отрицательных значений)
decrementBtn.addEventListener('click', function () {
  count = count - 1;
  render();
});
```

## Как исправить

Добавь проверку: уменьшать `count` только если он больше нуля.

```js
// ✅ Исправление
decrementBtn.addEventListener('click', function () {
  if (count <= 0) return;
  count = count - 1;
  render();
});
```

## Как лучше

Вариант с `if (count <= 0) return` предпочтительнее: он не вызывает лишний `render()` когда счётчик уже на нуле, и логика читается прямо — «если нельзя уменьшать, выходим сразу».
