# Баг 2 — Стрелка «вправо» ломается на последнем слайде

## Что происходит

На последнем слайде клик по стрелке «›» показывает пустое изображение (или ошибку в консоли).

## Почему так происходит

Индекс просто увеличивается на 1 без проверки границы:

```js
// ❌ Баг
nextBtn.addEventListener('click', function () {
  current = current + 1;
  show(current); // если current = 6, slides[6] = undefined — ошибка!
});
```

Массив `slides` содержит 6 элементов (индексы 0–5). При `current = 5` следующий шаг даёт `current = 6` — такого индекса нет.

## Как исправить

Используй оператор остатка от деления `%` — он «закольцовывает» счётчик:

```js
// ✅ Исправление
nextBtn.addEventListener('click', function () {
  current = (current + 1) % slides.length;
  show(current);
});
// При current = 5: (5 + 1) % 6 = 0 — возврат к первому слайду
```

## Как лучше

Оператор `%` (остаток от деления) — стандартный способ создать циклический счётчик. `(n + 1) % length` всегда остаётся в диапазоне `0..length-1`. Это работает для любого количества элементов и не требует `if`.
