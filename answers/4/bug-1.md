# Баг 1 — При загрузке нет картинки и подписи

## Что происходит

Страница открывается с пустым (сломанным) главным изображением и без подписи под ним.

## Почему так происходит

Функция `show(index)` существует и умеет отображать слайд, но нигде не вызывается при инициализации:

```js
// ❌ Баг — show(0) нигде не вызван
buildThumbs();
// страница загружена, но главное фото пустое
```

`mainImg.src` остаётся пустой строкой (как задано в HTML), потому что никто не попросил показать первый слайд.

## Как исправить

Вызвать `show(0)` после `buildThumbs()`:

```js
// ✅ Исправление
buildThumbs();
show(0);
```

## Как лучше

Это классический паттерн инициализации: написать функцию отображения и **не забыть вызвать её один раз при старте**. Хорошая привычка — после написания функции сразу думать: «А когда она должна вызваться впервые?»
