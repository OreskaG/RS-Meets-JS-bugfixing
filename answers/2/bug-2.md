# Баг 2 — Числа из полей склеиваются как строки

## Что происходит

Вместо арифметики получается конкатенация строк. Например, для счёта 1500 ₽ и 10% чаевые выходят 150010 вместо 150.

## Почему так происходит

`.value` у `<input>` всегда возвращает **строку**, даже если в поле введено число. Когда складываешь строки, JavaScript их склеивает:

```js
// ❌ Баг
const bill = billInput.value;       // "1500" — строка!
const tipPercent = tipInput.value;  // "10"   — строка!

const tipAmount = bill * tipPercent; // "1500" * "10" = 15000 (случайно работает, т.к. * принуждает к числу)
const total = bill + tipAmount;      // "1500" + 15000 = "150015000" — строка!
```

Оператор `*` неявно переводит строки в числа, а `+` — нет: если один из операндов строка, он делает конкатенацию.

## Как исправить

Явно преобразуй значения в числа сразу при чтении:

```js
// ✅ Исправление
const bill = Number(billInput.value);
const tipPercent = Number(tipInput.value);
const people = Number(peopleInput.value);
```

Запомни правило: **всегда явно конвертируй `input.value` в нужный тип** перед вычислениями — не полагайся на неявное преобразование JavaScript.
