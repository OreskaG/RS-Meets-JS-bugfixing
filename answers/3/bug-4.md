# Баг 4 — Клик по задаче не зачёркивает её

## Что происходит

При клике по тексту задачи ничего не происходит — зачёркивание не появляется.

## Почему так происходит

Обработчик клика меняет свойство `todo.done`, но после этого не вызывает `render()` для обновления DOM. Данные меняются, но страница об этом не знает:

```js
// ❌ Баг
li.addEventListener('click', function () {
  todo.done = !todo.done;
  // render() не вызван — DOM не обновляется
});
```

## Как исправить

```js
// ✅ Исправление
li.addEventListener('click', function () {
  todo.done = !todo.done;
  render(); // перерисовать список
});
```

## Как лучше

В этом приложении используется паттерн «данные → рендер»: сначала обновляем данные (`todos`), потом вызываем `render()` для синхронизации DOM с данными. Это простейший вариант реактивного подхода. Важно помнить: **изменение данных само по себе не обновляет страницу** — нужно явно вызвать отрисовку.
