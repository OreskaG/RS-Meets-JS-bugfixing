# Баг 3 — Кнопка «Удалить» не удаляет задачу

## Что происходит

При нажатии на «Удалить» задача остаётся на месте.

## Почему так происходит

Метод `splice(start, deleteCount)` принимает два аргумента: с какого индекса начать и **сколько элементов удалить**. Передан `0` вместо `1`:

```js
// ❌ Баг
todos.splice(i, 0);
// «Удали 0 элементов начиная с индекса i» — то есть ничего не удаляет
```

## Как исправить

```js
// ✅ Исправление
todos.splice(i, 1);
// «Удали 1 элемент начиная с индекса i»
```

После этого обязательно вызвать `render()`, чтобы обновить отображение.

## Как лучше

`splice` — универсальный и немного коварный метод: он может удалять, вставлять и заменять элементы одновременно. Второй аргумент (`deleteCount`) это именно **количество** удаляемых элементов, а не индекс конца. Для удаления одного элемента всегда передавай `1`.
